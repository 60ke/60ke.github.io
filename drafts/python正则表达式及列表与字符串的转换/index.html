re.findall与re.compile

测试代码

re.findall

import re
a = "wodiddjfkj"
b = re.findall(r"d.+?k",a)
c = re.findall(r"d(.+?)k",a)
print(b)
print(c)

输出：

['diddjfk']
['iddjf']

re.compile

import re
a = "wodiddjfkj"
b = re.compile(r"d.+?k")
c = b.findall(a)
print(b)
print(c)

输出：

re.compile('d.+?k')
['diddjfk']

补充

用(.+?)来匹配任意字符中“.”是不包含/n的所以。。。 解决办法： 用([\s\S]*)来表示

bb = re.findall("2\.([\s\S]*)",aa))

表示的是匹配aa中"2.“之后的所有内容
